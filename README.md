### 리액트 대시보드

## Installation
```
$ git clone https://github.com/wngml317/ts-react-dashboard

$ cd ts-react-dashboard
$ npm install 

$ cd client
$ npm install
```

## development (개발)
```
$ npm start
```

## production (배포)
```
$ cd client
$ npm run build

$ cd ts-react-dashboard
$ npm run server
```

## Version
```
chart.js      ::   3.9.1
react-scripts ::   4.0.3
```
