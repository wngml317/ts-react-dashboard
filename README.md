# 리액트 대시보드

## Installation
```
$ git clone https://github.com/wngml317/ts-react-dashboard
$ cd ts-react-dashboard
$ npm start
```

## Backend
```
$ npm install
```

## Frontend
```
$ cd client
$ npm install
```

## Build
```
$ cd client
$ npm run build
$ cd ..
$ npm run server
```
